pycharm-from-usb
================

Small script to:

1. Sets the Pycharm configuration path to the current drive letter the USB gets mounted when working on a different computer.
2. Launches Pycharm.

Usage:
================

Run
 
    start.bat
    
  
Requirements:
================

The folder structure of the USB must be

    /-
      /config
      /PyCharm Community Edition 3.1.1
